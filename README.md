# voska/tap

Homebrew formulae for CLI tools by [@voska](https://github.com/voska). Formulae are published automatically by [goreleaser](https://goreleaser.com/) on each tagged release.

## Installation

```bash
brew tap voska/tap
```

## Formulae

| Formula | Description |
|---|---|
| [amadeus](https://github.com/voska/amadeus-cli) | Amadeus Travel API CLI for humans and AI agents |
| [hn](https://github.com/voska/hn-cli) | Agent-friendly Hacker News CLI via the Algolia API |
| [qbo](https://github.com/voska/qbo-cli) | QuickBooks Online CLI for humans and AI agents |
| [toktap](https://github.com/voska/toktap) | Transparent proxy that taps LLM API traffic to extract token usage metrics |
| [zonasul](https://github.com/voska/zonasul) | Zona Sul supermarket CLI for AI agents |

## Usage

```bash
brew install voska/tap/<formula>
```
